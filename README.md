# cuneytinan.com

Personal portfolio of **Cüneyt İnan** — Frontend Developer based in Turkey.

🌐 **Live:** [cuneytinan.com](https://cuneytinan.com)

## About

Single-page portfolio built with vanilla HTML, CSS, and JavaScript. Features:

- Bilingual (TR/EN) with flag-based language toggle
- Animated contact ticker
- Custom DNA loader animation
- Adjustable ambient dimmer (preserves readability)
- Fully responsive layout

## Stack

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome (icons)
- No build step, no framework — pure static
